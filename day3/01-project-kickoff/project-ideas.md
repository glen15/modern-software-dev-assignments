# 프로젝트 아이디어

## 프로젝트 선정 기준

### 필수 요건

- [ ] 4시간 내 MVP 완성 가능
- [ ] 백엔드 API 2개 이상
- [ ] 프론트엔드 UI 존재
- [ ] Day 1-2 학습 내용 활용 가능

### 권장 요건

- [ ] 실용적인 기능
- [ ] 확장 가능성
- [ ] 팀원 모두가 관심 있는 주제

---

## 아이디어 목록

### 1. 할 일 관리 앱 (★ 입문자 권장)

**설명**: 스타터앱을 확장한 할 일 관리 앱

**핵심 기능**:
- 할 일 CRUD
- 완료 표시
- 우선순위 설정
- 마감일 알림

**확장 가능**:
- 카테고리 분류
- 검색 기능
- 통계 대시보드

**난이도**: ★☆☆

---

### 2. 팀 메모 공유 앱

**핵심 기능**:
- 메모 작성/수정/삭제
- 태그 기능
- 검색 기능
- 공유 링크 생성

**확장 가능**:
- 실시간 동기화
- 댓글 기능
- 마크다운 지원

**난이도**: ★★☆

---

- HTML/CSS
