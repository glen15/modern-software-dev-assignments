## 세션별 상세

### 킥오프 (01-project-kickoff/)

- 팀 구성 (3-4명)
- 프로젝트 아이디어 선정
- 역할 분담

### 스프린트 1: 기획 + 기본 구현

- 요구사항 정의
- 스타터앱 이해 및 수정
- 기본 백엔드 API 구현

### 스프린트 2: 핵심 기능 구현

- 프론트엔드 UI 구현
- 핵심 기능 완성
- AI 도구 적극 활용

### 스프린트 3: 보안 검토 + 테스트

- Semgrep 보안 스캔
- 취약점 수정
- 테스트 작성

### 데모 + 클로징

- 팀별 5분 발표
- 피드백 및 Q&A
- 워크샵 수료

---

## 심화 자료 (04-advanced/)

더 도전적인 프로젝트를 원하는 팀을 위한 자료입니다.

### 멀티 스택 가이드 (multistack/)
- Node.js/Express, Go/Gin 백엔드
- Next.js, React+Vite 프론트엔드
- 기술 스택 선택 가이드
- AI로 새로운 스택 빠르게 학습

### Bolt.new 가이드 (bolt-new/)
- AI 기반 풀스택 앱 빌더 활용
- 효과적인 프롬프트 작성법
- 생성된 코드 커스터마이징
- Claude Code와 조합 워크플로우
