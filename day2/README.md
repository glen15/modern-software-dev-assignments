## 세션별 상세

### 세션 1: 보안 기초 (01-security-intro/)

- OWASP Top 10 취약점 소개
- 실제 보안 사고 사례
- 개발자가 알아야 할 보안 기본 원칙

### 세션 2: Semgrep 보안 스캔 (02-semgrep-scan/)

- Semgrep 소개 및 설치
- 취약한 코드 스캔 실습
- 취약점 수정 실습

### 세션 3: 코드 리뷰 기초 (03-code-review/)

- 효과적인 코드 리뷰 방법
- 코드 리뷰 체크리스트
- 건설적인 피드백 작성법

### 세션 4: 수동 리뷰 + AI 보조 (04-testing-quality/)

- 수동 코드 리뷰 실습
- AI를 활용한 코드 리뷰
- 리뷰 의견 작성 연습

### 세션 5: 테스트 작성

- pytest 기초
- 단위 테스트 작성
- 테스트 커버리지

### 세션 6: 통합 과제 (assignments/)

- 취약점 탐지 + 수정 + 테스트 작성
- 난이도별 과제

워크샵 이후 더 깊이 학습하고 싶은 분들을 위한 자료입니다.

- 스택형 PR (Stacked PR) 워크플로우
- Graphite CLI로 PR 스택 관리

- 팀 협업 및 충돌 해결

## 사전 준비

1. Semgrep 설치:
   ```bash
   pip install semgrep
   # 또는
   brew install semgrep
   ```

2. pytest 설치:
   ```bash
   pip install pytest pytest-cov
   ```

자세한 환경 설정은 [SETUP.md](../SETUP.md)를 참고하세요.
