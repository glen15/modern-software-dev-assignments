# AI 기반 소프트웨어 개발 워크샵

## 개요

3일 집중 워크샵으로 AI 도구를 활용한 현대적 소프트웨어 개발 방법을 배웁니다.

**기간**: 3일 (24시간)
**대상**: 대학생, 교수, 공공기관 직원 (입문~심화 혼합)

---

## 학습 목표

워크샵을 마치면 다음을 할 수 있습니다:

- [ ] AI 도구를 활용하여 생산성 있게 코딩할 수 있다
- [ ] 효과적인 프롬프트를 작성할 수 있다
- [ ] 보안 취약점을 탐지하고 수정할 수 있다
- [ ] 코드 리뷰와 테스트를 통해 품질을 향상시킬 수 있다
- [ ] 팀으로 협업하여 프로젝트를 완성할 수 있다
