## 세션별 상세

### 세션 1: AI 활용 필요성 (01-introduction/)

- AI가 소프트웨어 개발에 미치는 영향
- 생산성 향상 사례와 통계
- AI 도구의 한계와 주의점

### 세션 2: 프롬프팅 기초 (02-prompting-basics/)

- **K-shot 프롬프팅**: 예시를 통한 학습 유도
- **Chain of Thought (CoT)**: 단계별 추론 유도
- **Self-Consistency**: 일관성 있는 답변 유도

### 세션 3: 프롬프팅 심화 (03-prompting-advanced/)

- **RAG (Retrieval-Augmented Generation)**: 외부 지식 활용
- **Tool Calling**: 외부 도구 호출
- **Reflexion**: 자기 반성을 통한 개선

### 세션 4: AI 코딩 도구 실습 (04-ai-coding-tools/)

- Cursor 사용법
- Claude Code 사용법
- 효과적인 활용 팁

### 세션 5: 개인 실습 과제 (assignments/)

- 입문자용 과제
- 심화자용 과제
