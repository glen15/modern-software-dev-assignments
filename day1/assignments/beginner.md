## 과제: 단어 퀴즈 프로그램

### 목표

영어 단어와 한글 뜻을 맞추는 간단한 퀴즈 프로그램을 만듭니다.

### 요구사항

1. 5개 이상의 영어 단어와 한글 뜻이 저장되어 있어야 합니다
2. 무작위로 단어를 선택하여 퀴즈를 출제합니다
3. 사용자 입력을 받아 정답 여부를 판단합니다
4. 5문제 출제 후 점수를 표시합니다

### 예시 실행

```
=== 단어 퀴즈 ===

문제 1/5: "apple"의 뜻은?
정답 입력: 사과
정답입니다! ✓

문제 2/5: "computer"의 뜻은?
정답 입력: 컴퓨터
정답입니다! ✓

문제 3/5: "book"의 뜻은?
정답 입력: 노트
오답입니다. 정답은 "책"입니다. ✗

...

최종 점수: 4/5 (80%)
```

## 단계별 가이드

### 1단계: 기본 구조 만들기

AI 도구에 다음과 같이 요청하세요:

```
Python으로 단어 퀴즈 프로그램의 기본 구조를 만들어줘.
- 영어 단어와 한글 뜻을 딕셔너리로 저장
- 5개 이상의 단어 포함
- main() 함수로 프로그램 시작
```

### 2단계: 퀴즈 로직 구현

```
이 프로그램에 퀴즈 기능을 추가해줘.
- random 모듈로 무작위 단어 선택
- 사용자 입력 받기
- 정답/오답 판단
- 5문제 출제
```

### 3단계: 점수 계산 추가

```
퀴즈 끝나면 점수를 계산하고 표시해줘.
- 맞은 개수 / 전체 문제 수
- 퍼센트로도 표시
```

### 4단계: 사용자 경험 개선

```
다음 기능을 추가해줘:
- 문제 번호 표시 (1/5, 2/5, ...)
- 정답/오답 시 이모지 추가 (✓, ✗)
- 틀렸을 때 정답 알려주기
```
